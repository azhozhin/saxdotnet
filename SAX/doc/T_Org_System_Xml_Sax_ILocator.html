Remarks

  • This interface combines features from the Locator and Locator2 interfaces specified in the original Java SAX API.
  • Any call on this interface before StartDocument or after EndDocument (that is, when not parsing) will throw a SaxException.
  • Because the definition of "parsing" in SAX is not restricted to textual document parsing, certain properties of ILocator may not be applicable in all contexts.
