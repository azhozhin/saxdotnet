Remarks

  • Optional method.
  • It is only legal to call Suspend when Status has the value Parsing, and this should only be done from within a handler call-back.
  • On return the Status property will have the value Suspended.
  • An implementation may decide to suspend parsing not immediately, but rather "as soon as possible". This can for instance mean that, after calling Suspend from the StartElement call-back of an empty element tag, the EndElement call-back may still follow.
