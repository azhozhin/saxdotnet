Classes

  Class Description
Compile Defines compile constants for interop with the Expat Dll.
Constants Defines constants for the Kds.Xml.Expat namespace.
EntityParseContext Holds entity specific information while the entity is being parsed. Can be subclassed to add more fields.
ExpatAttributes Expat specific IAttributes implementation.
ExpatException Base class for exceptions in Expat.
ExpatLocator Expat specific ILocator implementation.
ExpatParseError Expat specific ParseError implementation.
ExpatParseException Exception class for parse errors in Expat.
ExpatParser Object wrapper for Expat parser instance.
ExpatReader Wrapper class that exposes the (unmanaged) Expat XML parser through the IXmlReader interface.
ExpatStreamSource Expat specific subclass of InputSource.
ExpatUtils Provides helper routines for converting between managed and unmanaged data structures and adapting the Expat API for use with standard .NET framework classes. Also defines some Expat related constants. This is a static class (Module in Visual Basic)static classModule and so cannot be inherited or instantiated.
LibExpat This class wraps the the Expat library for interop purposes.
SaxEntityContext Class for internal use in ExpatReader. Groups all entity specific fields that are used during the parsing process.
SaxExpatParser Class for internal use in ExpatReader. Represents Expat parser.
StdEntityParseContext Convenience declaration, for use with StdExpatParser.
StdEntityParseContext Subclass of EntityParseContext that cooperates with StdExpatParser to resolve an external entity identifier, or an external subset request, to a Stream.
StdExpatParser Convenience declaration, so that a basic ExpatParser can be used without having to declare a subclass of EntityParseContext.
StdExpatParser Subclass of ExpatParser that provides entity resolver events together with a built-in default resolver.
StreamBufferReader Interop helper class for reading pinned data from Stream. To be used with ReadBuffer delegate type.
StringBufferReader Interop helper class for reading pinned data from string. To be used with ReadBuffer delegate type.
TextBufferReader Interop helper class for reading pinned data from TextReader. To be used with ReadBuffer delegate definition.
XMLMemoryHandlingSuite Represents the XML_Memory_Handling_Suite struct, which is used to pass memory handling function pointers to the Expat library.
